# فارسی
این صفحه مربوط به پروژه زیر هست، از این نسخه به بعد این پروژه بر پایه API ها گسترش پیدا خواهد کرد و پروژه قبلی دیگه به روز نخواهد شد. اولین نسخه این پروژه یعنی 2.6.0 همون ویژگی های پروژه قبلی با نسخه 2.5.10 رو دارا هست. و ویژگی های جدید که فقط در API ها قابل دسترس هست، به این پروژه کم کم افزونه خواهد شد. داکیومنت های این پروژه تا نسخه فعلی یعنی 2.6.0 رو از لینک پایین مطالعه کنید. تنها تفاوت نسخه 2.6.0 این پروژه با نسخه 2.5.10 پروژه قبلی فقط در دریافت اطلاعات ورودی هست. 2.5.10 با فایل های CSV و 2.6.0 از طریق API ها.
